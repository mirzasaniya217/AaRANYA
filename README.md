#  AaRANYA — Ethnic Fashion Experience

Live Website: https://mirzasaniya217.github.io/AaRANYA/

AARANYA is a theme-driven fashion web experience that blends **Indian ethnic elegance, cultural storytelling, and personalized styling**.  
The platform presents multiple fashion worlds and provides guided outfit suggestions based on **body type, skin tone, and skin texture**.

This project focuses on **design expression, fashion visualization, and structured web presentation** using pure HTML.

---

##  Project Vision

AARANYA represents the fusion of **Tradition + Personalization + Digital Experience**.  
It is designed for users who appreciate **ethnic, premium, and culturally inspired fashion**, while also exploring modern styling guidance.

---

##  Fashion Themes

The platform includes **7 unique fashion themes**, each reflecting a distinct styling mood and cultural identity:

-  **Raat Ke Rani** — Night elegance and luxury ethnic styling  
- **Devi** — Divine, graceful, and powerful traditional aesthetics  
- **Tana Bana** — Fabric storytelling and handcrafted textile inspiration  
-  Additional curated themes exploring ethnic fusion, cultural richness, and modern elegance  

Each theme is structured with its own visual identity and styling narrative.

---

## Personalized Styling Feature

AARANYA includes a guided outfit exploration section where styling is suggested based on:

- Body Type
- Skin Tone
- Skin Texture
- Personal Fashion Preference

This provides a **personalized and user-centric fashion experience**.
